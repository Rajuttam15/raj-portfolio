# Raj Uttam — Portfolio

A modern, animated personal portfolio built with React.js.

🌐 **Live:** https://Raj-Uttam.github.io/raj-portfolio

## Tech Stack

- React.js
- CSS-in-JS (inline styles)
- Particle canvas animations
- 3D tilt & cube effects
- Typewriter animation

## Sections

- Hero — Intro, stats, animated avatar
- Experience — IBM SkillsBuild internship
- Projects — Synvora, NIDAR, Blockchain, Donation System
- Skills — Proficiency bars & tech badges
- Contact — Social links & message form

## Run Locally
```bash
npm install
npm start
```

## Deploy
```bash
npm run deploy
```
